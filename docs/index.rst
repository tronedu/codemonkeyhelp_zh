簡介
===========================================
本人為 CodeMonkey 認證家教，在教學時常發現學生並非不懂程式的邏輯，只是因為對於英文的指令不熟悉，因此製作此文件希望更多的人不是因為這些問題卡關，讓更多人可以享受寫程式的樂趣。

*此文件為參考 CodeMonkey Help 的指令教學文件歡迎老師、家長及同學使用。
內容為個人學習分享，並非官方的正式文件，如有任何錯誤也請不吝指教。*
